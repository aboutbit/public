패밀리 베드
아이 침대 독립이란?

패밀리 베드처럼 붙이지 않고, 안방 안에 아이 전용 침대를 별도 공간에 독립 배치하는 방식.
부모는 기존 퀸 침대 그대로 유지, 아이는 같은 방의 다른 위치에서 잠.

02 — 현재 침대 구성 & 해결책
현

현재 구성

헤드보드 없음  +  받침대(박스스프링 또는 다이반)  +  퀸 매트리스

→ 받침대 높이 때문에 수면면이 높아 낙상 위험

→

해결책 — 받침대 빼기 (비용 0원)

받침대를 꺼내고 퀸 매트리스를 바닥에 내려놓기. 헤드보드가 없어 구조적으로 걸리는 것 없음.

받침대는 벽에 세워두거나 베란다/창고 보관.

✓

최종 구성 — 패밀리 베드 완성

바닥에 퀸 매트리스  +  옆에 슈퍼싱글 매트리스를 나란히.

가장 안전 + 비용 최소
02-1 — 두 매트리스 높이 맞추기

두 매트리스를 나란히 붙이려면 두께(높이)가 같아야 단차 없이 연결됩니다. 단차가 생기면 아이가 끼이거나 굴러 위험합니다.

① 퀸 두께 측정
받침대 뺀 후 퀸 매트리스 옆면을 줄자로 측정 (보통 20~30cm)
② SS 구매 기준
① 에서 잰 두께와 동일한 두께의 슈퍼싱글 선택
③ 차이 보정
2~5cm 차이는 낮은 쪽에 토퍼 얹어서 보정 가능
💡 베드 브릿지(틈새 패드)도 약간의 높이 차이를 보정하는 역할을 합니다.

04 — 안전 체크리스트
1

침대 높이 — 저상형 필수

18개월은 자면서 많이 움직입니다. 프레임 높이 20cm 이하 권장.

2

안전가드 (Safety Rail)

아이 쪽 열린 면에 레일 설치. 시중 제품 2~5만원대.

3

틈새 방지 — 베드 브릿지

퀸+SS 붙일 때 생기는 사이 간격 반드시 메우기.

⚠ 아이가 끼이면 매우 위험
4

매트리스 경도 — 탄탄한(Hard) 타입

패밀리 베드 구성 시 아이가 쓸 슈퍼싱글 매트리스를 Hard 경도로 선택. 너무 푹신하면 질식 위험.

05 — 구매 전략
당근 OK

침대 프레임

나중에 아이 방으로 옮길 거라 굳이 새거 살 필요 없음. 저상형 SS 프레임 당근 2~5만원대.

새것 추천

슈퍼싱글 매트리스

아이가 사용할 슈퍼싱글 매트리스. 위생·진드기 문제로 중고 비추. Hard 경도로 새것 구매. 10~20만원대.

새것 추천

베드 브릿지 / 틈새 패드

안전 관련 용품은 새것. 1~3만원대로 저렴.

당근 OK

안전가드 (세척 가능 제품)

세척 가능한 천 소재라면 중고도 무방. 알루미늄 프레임 제품 확인.

프레임은 당근으로 하고 매트리스만 새거 사면 될 것 같은데,
어차피 나중에 아이 방 독립할 때 그대로 옮기면 되니까 비용도 크게 안 들 것 같아.
당신은 아이 독방 언제쯤 분리하고 싶어?"
