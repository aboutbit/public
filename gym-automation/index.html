헬스장 음악 자동화

새벽 1시 음악 끄기 + PC 절전 / 새벽 5시 음악 자동 재생

Windows · 절전 모드

동작 방식

시간 동작
새벽 1:00 브라우저(크롬/엣지) 종료 → PC 절전 모드
새벽 5:00 PC 자동 해제 → 유튜브 전체화면 재생 (반복)
재생 영상 10시간짜리 헬스장 음악 — 영상이 끝나면 자동으로 처음부터 반복 재생됩니다.

파일 다운로드

setup_schedule.ps1 작업 스케줄러 자동 등록 스크립트 — 이것만 실행하면 됩니다
다운로드
gym_open.bat 새벽 5시 — 유튜브 자동 재생
다운로드
gym_close.bat 새벽 1시 — 브라우저 종료 + PC 절전
다운로드

설치 방법

1

파일 3개를 같은 폴더에 저장

위 파일 3개를 모두 다운로드하고 C:\gym-auto\ 폴더를 만들어 안에 넣어주세요.

2

스케줄 등록

setup_schedule.ps1 파일을 우클릭 → "PowerShell로 실행"

"완료!" 메시지가 뜨면 설치 완료입니다.

3

설치 확인 (선택)

  • Win + R 키 누른 후 taskschd.msc 입력 → 엔터
  • 왼쪽 "작업 스케줄러 라이브러리" 클릭
  • GymOpen, GymClose 작업이 보이면 정상
4

절전 해제 설정 확인

  • 작업 스케줄러에서 GymOpen 우클릭 → 속성
  • 조건 탭 → "이 작업을 실행하기 위해 컴퓨터의 절전 모드를 해제합니다" 체크 확인
참고 setup_schedule.ps1 실행 시 자동으로 설정됩니다. 혹시 체크가 안 되어 있다면 수동으로 체크해 주세요.

주의사항

PC를 완전히 끄면 안 됩니다 전원 버튼으로 완전 종료하면 새벽 5시에 자동으로 켜지지 않습니다. 반드시 절전(Sleep) 모드로 두세요.

마감 스크립트(gym_close.bat)가 자동으로 절전 모드로 전환하므로 별도로 조작할 필요는 없습니다.
문의사항이 있으시면 연락 주세요.
